DETAILLIERTE ANLEITUNG:

1) Installieren Sie vor der Codeeingabe bitte die Minddrops-App auf Ihrem Handy. Die App finden Sie hier:

2) Registrieren Sie sich bitte in der App. Sie können sich dort unkompliziert mit der auf Ihrem Gerät gespeicherten Google/Apple-ID oder auch per E-Mail und einem selbst gewählten Passwort registrieren.

3) Öffnen Sie diese Seite mit Ihrem Handy und melden Sie sich mit den bei der Registrierung verwendeten Daten an. Wenn Sie sich in der App per Google/Apple-ID angemeldet haben, dann können Sie hier nun erneut die entsprechenden Buttons verwenden.

4) Geben Sie auf der nächsten Seite dann bitte Ihren Gutscheincode ein und klicken Sie auf „Code einlösen“. Der von Ihnen eingegebene Code wird somit in der App aktiviert.

Sollten Sie den beiliegenden Code einer Audio-CD oder eines Buches eingelöst haben, wird Ihnen der dazugehörige Inhalt in der App auf Home/Meine Kollektionen angezeigt. Beachten Sie bitte, dass Motiv und Titel innerhalb der App abweichend von CD oder Buch sein können.

Sollten Sie einen Code für ein Minddrops-PLUS-Abo eingegeben haben, finden Sie dazu genauere Angaben in Ihrem User-Profil der App. Das Abonnement beginnt mit dem Datum der Codeeingabe. In der Regel werden diese Änderungen in der App direkt übernommen, sollte das nicht der Fall sein, schließen Sie bitte die App und starten sie dann neu.

Sollten Sie den beiliegenden Code einer Audio-CD oder eines Buches eingelöst haben, wird Ihnen der dazugehörige Inhalt in der App auf Home/Meine Kollektionen angezeigt. Beachten Sie bitte, dass Motiv und Titel innerhalb der App abweichend von CD oder Buch sein können. Sollten Sie einen Code für ein Minddrops-PLUS-Abo eingegeben haben, finden Sie dazu genauere Angaben in Ihrem User-Profil der App. Das Abonnement beginnt mit dem Datum der Codeeingabe. In der Regel werden diese Änderungen in der App direkt übernommen, sollte das nicht der Fall sein, schließen Sie bitte die App und starten sie dann neu.
